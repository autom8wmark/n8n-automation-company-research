# Company Research Workflow (n8n + OpenAI)

This workflow automates company research process using **n8n**, **OpenAI**, **SerpAPI**, **Airtable** and **Google Sheets**.

## Features
- Search for company information
- Extract website data
- Create tasks for AI
- Scrape the website
- Update database in Airtable

## Tools & APIs
- n8n (workflow automation)
- OpenAI API (Whisper for transcription)
- Google Drive API
- Google Sheets API
- SerpAPI
- Airtable API

## Files
| File | Description |
|------|--------------|
| `Company Research Public.json` | Import this workflow into n8n |
| `Company Research Public.docx` | Full setup guide and explanation |

## Usage
1. Import the `.json` file into your n8n instance.
2. Add your OpenAI,Google API, Airtable, SerpAPI credentials.
3. Adjust Google Sheets folder and file names.
4. Run the workflow!
